### Install

#### Clone this repo

```
git clone https://github.com/jkitsao/angular-github-api.git
```

#### Install dependencies

```
npm install
```

or

```
yarn
```
### Build for production

```bash

ng build
```
### Usage

#### Run the app

```
ng serve -o
```
open your browser on localhost:4200
